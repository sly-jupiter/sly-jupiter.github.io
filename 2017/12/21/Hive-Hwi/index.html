Hive用户接口(HWI)

发表于 2017-12-21 | 分类于 大数据 ， Hive

Hive提供给用户的交互接口一般有三种：Hive命令行(Hive Command Line)、JDBC/ODBC以及Hive WEB Interface（HWI）

本文介绍Hive WEB Interface（HWI）

配置

HWI的运行需要依赖两个包：hive-hwi-0.13.1.jar和hive-hwi-0.13.1.war
这两个包应该都部署在$HIVE_HOME/lib目录下

但Hive0.13.1的安装包lib目录下没有提供hive-hwi-0.13.1.war，
所以要下载Hive0.13.1的源码，进入源码的\hwi\web\目录下，

将该目录下的文件夹和文件压缩成zip包，

再重命名为hive-hwi-0.13.1.war，放到$HIVE_HOME/lib目录下即可。

配置$HIVE_HOME/conf/hive-site.xml

再做一步，防止报错
